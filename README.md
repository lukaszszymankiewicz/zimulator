Zimulator

Game Boy Emulator in Zig

Right now it is just a rudimentary PoC that can run only one program (Hello World).

![emulator working](screenshot.png)

Hello world program used from: https://gbdev.io/gb-asm-tutorial/index.html

Game Boy instructions table: https://meganesu.github.io/generate-gb-opcodes/

Ton of Knowledge: https://gbdev.io/pandocs/
